Overview

What Is KeyStatus?

KeyStatus does a very basic job. It acts as virtual LED lights for Caps Lock and so forth. For example It displays a red notification icon with the letter 'C' in red when Caps Lock is turned on, when Caps Lock is activated however, the notification icon of the letter 'C' will go green. KeyStatus also comes with a feature called Active Keys active keys shows a little dialog containing the status of the lockable keys. It will display something like this Caps Lock: Active this text's fore color will be set to green when that key is locked and will change its fore color to red when it is not locked.

How Did This Become?

It was pretty simple, I was out shopping one day and came across this Microsoft keyboard and mouse set, so being me, I bought it. Later that day when I came home I plugged it in but yet to notice, it had no LED lights for Caps Lock etc. So sometimes I would be typing yet I didn't know that Caps Lock was turned on, which was quite annoying. Therefore I decided to develop KeyStatus.

Compatibility

KeyStatus should work on 32-Bit and 64-Bit Windows machines (WinXP and Up). It will have compatibility on 64-Bit machines since KeyStatus is compiled on a 32-Bit machine this means that WOW64 (Windows on WIndows 64) will then obviously be in use.
